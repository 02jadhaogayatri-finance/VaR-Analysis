# 📊 Value at Risk (VaR) Analysis – Asian Paints

This project focuses on measuring the market risk of Asian Paints stock using two widely used Value at Risk (VaR) methodologies:

- Historical Simulation Approach  
- Monte Carlo Simulation Approach  

The objective of this analysis was to estimate the potential downside risk in the stock price under different confidence levels and compare how different risk models interpret market uncertainty.

## 🚀 Objective
To analyze and compare downside risk estimates of Asian Paints stock using historical market data and probability-based simulation techniques.

## 📈 Methodology

### 🔹 1. Historical Simulation Method
- Collected historical stock price data of Asian Paints  
- Calculated daily returns and return distribution  
- Estimated VaR at:
  - 90% confidence level
  - 95% confidence level
  - 99% confidence level
- Used historical market movements to estimate downside risk

### 🔹 2. Monte Carlo Simulation Method
- Simulated future stock return scenarios using probability distributions  
- Generated random return outcomes based on mean and standard deviation  
- Calculated VaR using simulated market scenarios  
- Compared simulation-based results with historical VaR estimates

---

## 📊 Key Findings

### Historical Simulation VaR
| Confidence Level | VaR % |
|---|---|
| 90% | -1.67% |
| 95% | -2.41% |
| 99% | -4.23% |

### Monte Carlo Simulation VaR
| Confidence Level | VaR % |
|---|---|
| 90% | -1.96% |
| 95% | -2.58% |
| 99% | -3.72% |

## 🔍 Key Insight
The Monte Carlo Simulation approach produced slightly different downside risk estimates compared to the Historical Simulation method, highlighting how model assumptions and simulated market scenarios can influence risk measurement.
